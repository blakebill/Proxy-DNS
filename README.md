# Proxy-DNS
### Get the fastest domain name resolution.

##### DNS_Global is mainly used for most users worldwide.

##### DNS_UnGlobal is mainly used for mainland users.

## DNS_UnGlobal Profiles

- `DNS_UnGlobal_Surge.conf`
- `DNS_UnGlobal_Loon.conf`
- `DNS_UnGlobal_Shadowrocket.conf`
- `DNS_UnGlobal_QuantumultX.conf`

The four profiles are generated from the same curated split-DNS domain set. CN domains use CN resolvers, while global/proxy-prone domains use global resolvers.

`DNS_UnGlobal.conf` is kept as the legacy source reference.
